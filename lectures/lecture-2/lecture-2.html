Supervised Learning

Basic idea

  • We have a set of labelled data
    • {(x1,y1),...,(xn,yn)}{(x1,y1),...,(xn,yn)}
  • We assume there is some function
    • F(X):X→YF(X):X→Y
  • The goal of Supervised Learning is to find (from the examples)
    • g(θ,X):X→Yg(θ,X):X→Y
  • such that g(θ,X)g(θ,X) approximates F(X)F(X) Supervised because we can compare g(θ,X)g(θ,X) to YY

Supervised Learning

Training (Supervised learning)

  • Ideally, we want to approximate F(X):X→YF(X):X→Y for all XX
  • But, for now, we'll consider only our Training Set
    • {(x1,y1),...,(xn,yn)}{(x1,y1),...,(xn,yn)}
  • Training Set
    • The data we use to adjust the parameters θθ in our model.
    • More generally: data used to choose a hypothesis
  • Training Error or Empirical Error
    • The error on the training set for each instance of θθ.
    • (Sample Error in Mitchell 1997)

Supervised Learning

Our ML problem for today:

  • Goal: Predict the YY values in our training set
  • Performance: minimise training error
  • Data: {(x1,y1),...,(xn,yn)}{(x1,y1),...,(xn,yn)}

Classification and Regression

  • In Classification YY is discrete.
    • Examples: SPAM detection, predict if mushrooms are poisonous
    • Find function to split data in differente sets
  • In Regression YY is continuous.
    • Examples: predicting trends, prices, purchase probabilities
    • Find function that approximates YY

Likelihood

  • Data points (x,y)(x,y) are randomly sampled from all possible values.
  • But θθ is not a random variable.
  • Find the θθ that, if true, would make the data is most probable
  • In other words, find the θθ of maximum likelihood

import numpy as np
import matplotlib.pyplot as plt

mat = np.loadtxt('polydata.csv',delimiter=';')
x,y = (mat[:,0], mat[:,1])
coefs = np.polyfit(x,y,2)

pxs = np.linspace(0,max(x),100)
poly = np.polyval(coefs,pxs)

plt.figure(1, figsize=(12, 8), frameon=False)
plt.plot(x,y,'or')
plt.plot(pxs,poly,'-')
plt.axis([0,max(x),-1.5,1.5])
plt.title('Degree: 2')
plt.savefig('testplot.png')
plt.close()

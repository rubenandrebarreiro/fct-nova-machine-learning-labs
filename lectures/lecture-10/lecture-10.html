Kernel Trick

Kernel trick

  • With an SVM we only need the inner products of the examples:
    • argmaxα⃗ ∑n=1Nαn−12∑n=1N∑m=1Nαnαmynymx⃗ Tnx⃗ margmaxα→∑n=1Nαn−12∑n=1N∑m=1Nαnαmynymx→nTx→m
    • (as we did with the H matrix).

The kernel trick

  • A kernel function is a function of two vectors whose result is the inner product of some function of each vector:
    • K(u⃗ ,v⃗ )=ϕ(u⃗ )Tϕ(v⃗ )K(u→,v→)=ϕ(u→)Tϕ(v→)
  • Useful if K(u⃗ ,v⃗ )K(u→,v→) is easier to compute than ϕ(u⃗ )Tϕ(v⃗ )ϕ(u→)Tϕ(v→)
  • For example ϕ2(x⃗ )=[1,2√x1,2√x2,2√x1x2,x21,x22]Tϕ2(x→)=[1,2x1,2x2,2x1x2,x12,x22]T
    • ϕ2(u⃗ )Tϕ2(v⃗ )=(u⃗ Tv⃗ +1)2ϕ2(u→)Tϕ2(v→)=(u→Tv→+1)2
  • More generally, Kϕn(u⃗ ,v⃗ )=(u⃗ Tv⃗ +c)nKϕn(u→,v→)=(u→Tv→+c)n

Kernel trick

  • So now we solve:
    • argmaxα⃗ ∑n=1Nαn−12∑n=1N∑m=1NαnαmynymK(x⃗ n,x⃗ m)argmaxα→∑n=1Nαn−12∑n=1N∑m=1NαnαmynymK(x→n,x→m)
  • Where K(x⃗ n,x⃗ m)K(x→n,x→m) is the kernel function for some non-linear expansion ϕϕ of our original data
  • To classify new points, we do not compute w⃗ w→ or w0w0, because the hyperplane is in the space of ϕ(x⃗ )ϕ(x→)
  • Instead, we compute the class of xt→xt→ from the support vectors:
    • w⃗ Tϕ(xt→)+w0=∑n=1NαnynK(xn→,xt→)w→Tϕ(xt→)+w0=∑n=1NαnynK(xn→,xt→)

Kernel trick

  • Example, using a polynomial kernel of degree dd:
    • Kϕ(x⃗ d)(x1→,x2→)=(x1→Tx2→+1)dKϕ(x→d)(x1→,x2→)=(x1→Tx2→+1)d
def H_poly(X,Y,d):
    H = np.zeros((X.shape[0],X.shape[0]))
    for row in range(X.shape[0]):
        for col in range(X.shape[0]):
            k = (np.dot(X[row,:],X[col,:])+1)**d
            H[row,col] = k*Y[row]*Y[col]
    return H
  • Evaluating a point:
    • w⃗ Tϕ(xt→)+w0=∑n=1NαnynKϕ(x⃗ d)(xn→,xt→)w→Tϕ(xt→)+w0=∑n=1NαnynKϕ(x→d)(xn→,xt→)
def poly_k_class(X,alphas,Y,xt,n):
    s = 0
    for ix in range(len(alphas)):
        s = s + (np.dot(X[ix,:],xt)+1)**n*Y[ix]*alphas[ix]
    return s
